- To run SEAL Demolition Derby controller:

Enter bin directory via command line and run:

$java -cp .:bin:lib/jFuzzyLogic.jar sealbot.Client sealbot.SealBot host:localhost port:<port> id:championship2012 maxEpisodes:<maxEpisodes> maxSteps:<maxSteps>

The port, maxEpisodes and maxSteps must be specified.
